# Privacy Policy — SF-Industries Error Inspector

**Effective date:** 2026-05-28

## Summary

SF-Industries Error Inspector does **not** collect, store, transmit, or share any personal data or user information of any kind.

## What the extension does

SF-Industries Error Inspector is a Chrome DevTools extension that reads network responses from the Salesforce Industries (Vlocity) Aura endpoint within the user's own active browser session. It displays the parsed content directly inside the Chrome DevTools panel.

- All processing happens **locally in your browser**.
- No data is sent to any external server.
- No data is stored beyond the current DevTools session (data is cleared when DevTools is closed or the panel is reset).
- The extension does not use analytics, telemetry, or any third-party services.

## Permissions

The extension requests no Chrome permissions beyond the standard DevTools panel API (`devtools_page`), which is scoped solely to displaying a panel inside Chrome DevTools.
